# Predi-o-de-Valores-Imobili-rios---End-to-End-Machine-Learning
Este repositório contém o desenvolvimento de um projeto completo de Machine Learning, baseado na metodologia estruturada no livro Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow de Aurélien Géron.  O objetivo principal é prever o preço médio de residências em distritos da Califórnia
